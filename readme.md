### **FinTrack CUCEI - Asistente Inteligente de Finanzas Personales** 

**Objetivo General:**:

Desarrollar una aplicación móvil Android y un sistema backend distribuido en la nube que permita a los usuarios registrar ingresos y gastos, visualizar su situación financiera, y obtener predicciones de gastos futuros utilizando un modelo de Machine Learning (Regresión Lineal).

**¿Por qué este proyecto?**

Utilidad real: Resuelve un problema cotidiano (control financiero).

Complejidad técnica adecuada: No es trivial, pero es alcanzable en 4.5 meses.

Cumple al 100% los 3 módulos: BD (M2), Distribuido (M3), ML (M4).

Diferenciador: El enfoque en predicción con ML y la arquitectura distribuida con hilos en Android.
